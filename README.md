# Cosmetics Management System

This project is a database management system designed to manage a cosmetics store efficiently.

## Features
- Product management
- Customer management
- Order processing
- Inventory tracking
- Payment management
- Customer reviews

## Technologies Used
- SQL
- HTML
- CSS
- JavaScript
- Microsoft SQL Server
