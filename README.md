Cosmetics Management System database project developed using SQL for managing products, customers, orders, inventory, and payments in a cosmetic store.

project description:

Cosmetics Management System is a database-driven project designed to manage cosmetic store operations efficiently. The system stores and organizes information related to customers, products, orders, suppliers, inventory, and payments using SQL.

The database structure includes multiple entities such as Customer, Product, Category, Brand, Order, Supplier, Purchase, Inventory, Employee, Payment, and Review. These tables are connected using primary and foreign key relationships to ensure proper data organization.

The system supports operations such as adding products, managing stock levels, processing orders, tracking payments, and maintaining customer records. This project demonstrates concepts of relational database design, ER modeling, SQL queries, and database management.
